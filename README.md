# alessiocallaioli.github.io
This repository contains multiple projects showcasing different technologies, patterns, and skills. Each project is organized in its own folder and is designed to be easy to run and extend.

General Recommendations
	•	Use Java 17+ for Spring Boot projects
	•	Ensure MySQL 8+ is running locally for database projects
	•	For the JSP/Tomcat project, ensure Tomcat 10+ is installed and configured

  
Projects overview:


1. User Management API
	•	Stack: Spring Boot + MySQL
	•	Description: RESTful API for managing users with CRUD operations, input validation, and modular architecture.
	•	Folder: user-management/


2. Small Commerce Backend
	•	Stack: Spring Boot + MySQL
	•	Description: Backend for a small e-commerce application, including user, product, and order management.
	•	Folder: small-commerce/


4. Dashboard JDBC Web App
	•	Stack: JSP/Servlets + Tomcat + MySQL + JDBC
	•	Description: Simple Java web application demonstrating a login dashboard with session management.
	•	Folder: dashboard-jdbc/
